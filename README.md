# TF - DW - Aula 03 - Node.js: Tipos de Referência e loops

## Node.js: Tipos de Referência e Loops

Este repositório contém o Trabalho Final da Aula 03 da disciplina de Desenvolvimento Web.

O objetivo do exercício é demonstrar, em um único arquivo JavaScript executável no Node.js, os principais conceitos relacionados a tipos de referência em JavaScript.

## Objetivo

Criar um objeto principal contendo propriedades simples e um array interno, e utilizar esse objeto para demonstrar:

- criação e uso de objetos
- uso de array dentro de objeto
- iteração com `for...in`
- iteração com `for...of`
- comparação entre tipos de referência
- cópia por referência
- cópia utilizando spread operator
- transformação do objeto em JSON com `JSON.stringify()`
